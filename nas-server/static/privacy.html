MarkItDown

Это self-hosted приложение — MarkItDown работает на инфраструктуре, которую контролируешь ты сам (свой сервер, VPS, домашний NAS или просто локальный компьютер), а не в чьём-то облаке. Файл конвертируется прямо там же и никуда не отправляется во внешние сервисы. Результат сохраняется в архив на этом же сервере — не у разработчика и не в чужом облаке. Удалить любой файл из архива можно в любой момент. Код открыт: github.com/serjdrej/markitdown-desktop.

This is self-hosted software — MarkItDown runs on infrastructure you control yourself (your own server, a VPS, a home NAS, or just your local machine), not in someone else's cloud. The file is converted right there and never sent to any external service. The result is saved to the archive on that same server — not on the developer's servers, not in any third-party cloud. You can delete any file from the archive at any time. The source code is public: github.com/serjdrej/markitdown-desktop.
